4. Getting Started: Setup Guide

Step 1: Flashing the Firmware

Connect the FC to your computer using a USB cable. Open the Betaflight Configurator and navigate to the "Firmware Flasher" tab. Select the correct board target and load the hex file you downloaded. Click "Flash Firmware".

Step 2: Initial Configuration

After flashing, reconnect to the FC. Go through the initial setup wizard to calibrate the accelerometer and set up your receiver.
